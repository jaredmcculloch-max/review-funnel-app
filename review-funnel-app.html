Setup Guide
Follow these steps to go live — takes about 45 minutes total
✓
1. Get your Google Review link
Search your business name on Google. On the right side panel, click "Get more reviews." Copy that link. Then paste it into Settings → Business Info → Google Review link.
2
2. Create a Twilio account
Go to twilio.com and sign up for a free account. Once in, go to Phone Numbers → Buy a number (about $1/month). Then go to Account → API Keys and copy your Account SID and Auth Token. Paste both into Settings → Twilio. You'll also need to complete Twilio's A2P registration (free, takes 1–3 days) to send texts in the US.
3
3. Connect to Stripe
Log into stripe.com → Developers → Webhooks → Add endpoint. Paste in the webhook URL from Settings → Stripe section. Select the event payment_intent.succeeded. Copy the Webhook Signing Secret and your Secret Key into Settings → Stripe.
4
4. Host this app
This app needs to live on a server so Stripe can send it data. The easiest free option is Railway.app or Render.com. Create a free account on either, upload this file, and they give you a public URL. Copy that URL and update the webhook URL in Settings → Stripe, then re-add it in Stripe's dashboard. If this feels intimidating, ask a local tech-savvy person or post on Fiverr — deploying this file takes about 10 minutes for someone who knows how.
5
5. Send a test text
Once all credentials are saved, go to the Dashboard and use the "Test Your Setup" panel to send a text to your own phone number. Confirm it arrives and that replies trigger the correct follow-up. You're live!
